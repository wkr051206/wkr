##  函数
### 函数的四种形式
1. 无参数 无返回值
```c
void fun(){ // fun 是函数名称
	....
	// 无return
}
```
2. 无参数，有返回值
```c
int fun(){  // int是因为返回的变量是一个int类型的
 ....
 return a; // a是返回的变量
 }
```
3. 有参数 无返回值
   ```c
   void fun(int a,double b){ 
	   //fun是函数的名称
	   //int 是参数a的类型 ，double是参数b的类型
	   ....
   }
   
   
   f(x) = 2x-1
   f 就是函数
   x 就是参数
   x可以定义在整数（int），也可以定义在实数（double）
   ```
3. 有参数，有返回值
   ```c
   int fun(int n){
   int sum = 0;
   ....
   return sum;
   }
   ```

## 数组
 - 内存地址：&
 ```c
 .#include<stdio.h>
 int main(int argc,char const *argv[])
 {
 int a=5;
 int b=10;
 printf("%p\n",&a);//p就表示打印出地址
 printf("%p\n",&b);
 return 0;
 }
 ```
 - 数组与sizeof
 sizeof的用法是显示出数组在内存中占多少字节。
  ```c
.#include<stdio.h>
 int main(int argc,char const *argv[])
 {
 int a=5;
 int b=10;
 printf("%zu\n",sizeof(a));//p就表示打印出地址
 printf("%zu\n",sizeof(b));
 printf("%zu\n",sizeof(3.14))
 return 0;
 }
 ```
 数组循环：
 ```c
 for(int i=0;i<sizeof(a)/sizeof(a[0]);i++)
 //只要索引 `i`小于数组的长度，循环就继续。
 ```

## 指针
指针是用来存放内存地址的变量。
- 指针的声明
```c
int a;//声明一个整形变量
int *p//声明一个指针变量，该指针指向一个int类型的地址
```
![](media/Pasted%20image%2020251003232444.png)
```c
int a=5;
int *p=&a;
printf("%d\n",*p);//此时会输出5
*p=100;
printf("%d\n",a);//修改了a所在的地址
```
