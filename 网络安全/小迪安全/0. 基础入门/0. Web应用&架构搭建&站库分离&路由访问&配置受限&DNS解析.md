## 常规的Web应用搭建
1. 购买云服务器，购买域名（阿里云）
2. 云服务器去搭建中间件（他说在网上搜一下怎么搭建（比如Windows搭建iis））
3. 下载并上传Web程序源码
4. 添加网站并绑定域名目录
	1. IIS管理器中右键网站，选择添加网站![](media/Pasted%20image%2020250121224539.png)
	2. 输入名称，选择路径![](media/Pasted%20image%2020250121224927.png)路径指向网站源码目录（随便在网上下载一个源码）下载后上传
	3. 选择类型，IP地址，主机名就是域名![](media/Pasted%20image%2020250121225320.png)
	4. 添加记录![](media/Pasted%20image%2020250127154312.png)IP地址填这个![](media/Pasted%20image%2020250127154422.png)
	5. 输入主机名（主机记录＋名称）![](media/Pasted%20image%2020250127155508.png)点击确定
输入同一IP地址网站名称访问网站时，由于没有绑定主机名，显示的是访问IP的内容。如果想访问。可以添加网站绑定![](media/Pasted%20image%2020250127161600.png)![](media/Pasted%20image%2020250127161844.png)![](media/Pasted%20image%2020250127163803.png)

## 站库分离
一种在网站或网络应用开发中常用的架构模式，它的主要思想是将网站的前端（即用户看到的和与之交互的界面部分，通常称为“站”）与后端的数据存储（即数据库，简称“库”）分开部署
