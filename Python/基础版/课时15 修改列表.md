## 列表(List)
- List是可变的，它是序列
- 在方括号中添加元素，并使用逗号隔开
```python
list0 = []#空列表
list1 = ['China'.1997,2000]
list2 = [1,2,3,4,5]
list3 = ["a","b","c","d"]
list4 = ['red','green']
```

## 修改列表
列表是可变的，我们可以通过索引和切片的方式来对列表的元素进行修改
```python
list1 = ["h","e","l","l","o"," ","1","牛","3"]
print(len(list1))
list1[-2]="羊" #列表可变，涉及增改操作时，通常会对原数据进行操作
```
