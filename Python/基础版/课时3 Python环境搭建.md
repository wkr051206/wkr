## IDE安装
下载python之后在命令行写代码不方便，可以用IDE（集成开发环境）来写代码。
主流的python IDE有：Vim,VSCode,PyCharm,Jupyter等，可以根据个人习惯选择不同的IDE。
根据课程先选择VSCode。
