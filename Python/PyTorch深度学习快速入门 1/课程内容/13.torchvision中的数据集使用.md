**核心：用`torchvision.datasets`加载数据集，搭配 `DataLoader`实现自动分批/打乱/并行加载。​**

## 使用步骤
1. 导入工具
```python
from torchvision import datasets, transforms
from torch.utils.data import DataLoader
```
2. 设置数据预处理​（关键！）
```python
transform = transforms.Compose([
     transforms.ToTensor(),          # 将图像转成Tensor格式
     transforms.Normalize((0.5,), (0.5,)) # 归一化（可选）
])
```
3.  加载数据集​
```python
# 下载训练集（若download=True，首次运行自动下载）
train_data = datasets.CIFAR10(
     root='./data',                  # 数据集保存路径
     train=True,                     # True=训练集, False=测试集
     transform=transform,            # 应用预处理
     download=True                   # 自动下载
)

# 创建数据加载器（自动分批打乱）
train_loader = DataLoader(
     train_data,
     batch_size=64,                 # 每批加载64张图片
     shuffle=True                    # 打乱数据顺序
)
```

## 立即使用示例
```python
# 遍历数据（直接复制可用）
for images, labels in train_loader:
    # images: [64, 3, 32, 32] 的张量 (批次, 通道, 高, 宽)
    # labels: [64] 的标签张量
    print("一批图像形状:", images.shape)
    print("对应标签:", labels)
    break  # 查看第一批即可
```
