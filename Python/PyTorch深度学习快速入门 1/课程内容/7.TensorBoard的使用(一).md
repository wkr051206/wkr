## SummaryWriter类使用

![](media/Pasted%20image%2020250811222500.png)

![](media/Pasted%20image%2020250811222559.png)
创建一些示例，存储到文件夹下
![](media/Pasted%20image%2020250811222830.png)
![](media/Pasted%20image%2020250811222859.png)
```python
from torch.utils.tensorboard import SummaryWriter  # 导入TensorBoard记录器

# 创建记录器实例（日志将保存在"logs"目录）
writer = SummaryWriter("logs")

# 记录图像数据（需要添加具体参数）
writer.add_image()

# 记录标量数据（如损失值、准确率）
writer.add_scalar()

# 关闭记录器（确保所有数据写入磁盘）
writer.close()
```
![](media/Pasted%20image%2020250812110734.png)
```python
# 完整示例
for epoch in range(epochs):
    # ...训练代码...
    loss = calculate_loss()
    acc = calculate_accuracy()
    
    # 记录到TensorBoard
    writer.add_scalar('Loss/train', loss, epoch)
    writer.add_scalar('Accuracy/train', acc, epoch)
    
    # 每10个epoch记录一次图像
    if epoch % 10 == 0:
        sample_images = get_sample_outputs()
        writer.add_image('Predictions', sample_images, epoch)
```
