## 有序地管理环境
![](media/Pasted%20image%2020250705103112.png)
不同项目也许需要用到不同的环境，所以需要有序的管理环境。
我们首先使用conda指令创建一个屋子，名叫pytorch。


创建conder环境：
1. 在开始菜单中寻找anaconda prompt并打开
2. 输入指令
   `conda create -n pytorch-latest python=3.13.5`
   -n：name首字母
   pytorch-latest：名称
   python=3.11：环境
   （如果需要删除已经存在的环境，输入命令conda env remove -n pytorch，并选择yes）
   ![](media/Pasted%20image%2020250705173928.png)
3. 输入y表示yes
   ![](media/Pasted%20image%2020250705222900.png)
4. 输入activate pytorch-latest
   ![](media/Pasted%20image%2020250706000652.png)
   会发现前面括号内的后缀由base变成了pytorch-latest，即指令的环境的名称。
5. 查询房间中有哪些工具包，输入pip list
   ![](media/Pasted%20image%2020250706002845.png)
   发现没有pytorch，去安装
6. 找到合适选项，输入相应命令并安装
   `pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118`  ![](media/Pasted%20image%2020250707235210.png)
7. 检验是否安装成功
   输入python进入检验
   先输入`import torch`检验，没有问题
   输入`torch.cuda.is_available()`,返回True
   环境配置完成
   ![](media/Pasted%20image%2020250708123019.png)

成功
