为什么需要这些转换？
1. 统一标准​​：不同库使用不同数据格式，转换确保格式一致

2. GPU加速​​：Tensor格式可以直接在GPU计算

3. 数值稳定​​：归一化0-1范围防止梯度爆炸

4. 效率提升​​：(C,H,W)内存排列更符合GPU读取方式

举例：
```python
# PIL → Tensor（最常用）
from PIL import Image
from torchvision import transforms

pil_img = Image.open("image.jpg")
tensor_img = transforms.ToTensor()(pil_img)  # 得到(C, H, W)的Tensor

# OpenCV → Tensor（需要额外处理）
import cv2

cv_img = cv2.imread("image.jpg")
cv_img = cv2.cvtColor(cv_img, cv2.COLOR_BGR2RGB)  # 必须转RGB
tensor_img = transforms.ToTensor()(cv_img)
```
