## 二、卷积神经网络架构设计

### 基础CNN结构（带维度计算）

```python
import torch.nn as nn

class CIFAR10CNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.features = nn.Sequential(
            # 输入: [3, 32, 32]
            nn.Conv2d(3, 32, kernel_size=3, padding=1),  # 输出: [32, 32, 32]
            nn.ReLU(),
            nn.MaxPool2d(2),  # 输出: [32, 16, 16]
            
            nn.Conv2d(32, 64, kernel_size=3, padding=1),  # 输出: [64, 16, 16]
            nn.ReLU(),
            nn.MaxPool2d(2),  # 输出: [64, 8, 8]
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),  # 输出: 64 * 8 * 8 = 4096
            nn.Linear(4096, 512),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(512, 10)
        )
    
    def forward(self, x):
        x = self.features(x)
        x = self.classifier(x)
        return x
```

### 卷积输出尺寸计算公式

```python
def conv_output_size(H_in, W_in, kernel_size, stride=1, padding=0, dilation=1):
    H_out = (H_in + 2*padding - dilation*(kernel_size-1) - 1) // stride + 1
    W_out = (W_in + 2*padding - dilation*(kernel_size-1) - 1) // stride + 1
    return H_out, W_out
```

## 三、Sequential模块高级应用

### 模块化构建技巧

```python
def conv_block(in_channels, out_channels, pool=True):
    layers = [
        nn.Conv2d(in_channels, out_channels, kernel_size=3, padding=1),
        nn.BatchNorm2d(out_channels),
        nn.ReLU(inplace=True)
    ]
    if pool:
        layers.append(nn.MaxPool2d(2))
    return nn.Sequential(*layers)

model = nn.Sequential(
    conv_block(3, 32),  # 32x16x16
    conv_block(32, 64),  # 64x8x8
    conv_block(64, 128), # 128x4x4
    conv_block(128, 256, pool=False), # 256x4x4
    nn.AdaptiveAvgPool2d(1),  # 256x1x1
    nn.Flatten(),
    nn.Linear(256, 10)
)
```

## 五、网络调试与优化技巧

### 常见错误排查表

|错误现象|可能原因|解决方案|
|---|---|---|
|输出尺寸不匹配|展平后维度计算错误|使用`validate_model`验证各层输出|
|训练不收敛|学习率不当/未归一化|使用Adam优化器，添加BatchNorm|
|过拟合|模型复杂度过高|增加Dropout，添加L2正则化|
|梯度爆炸|未使用梯度裁剪|`nn.utils.clip_grad_norm_(model.parameters(), 1.0)`|
|内存不足|批次过大|减小batch_size，使用梯度累积|
